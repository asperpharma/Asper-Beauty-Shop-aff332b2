# Security Policy

## Reporting a Vulnerability

If you believe you have discovered a security vulnerability in this project, we
encourage you to report it confidentially through email. Please do not disclose
it publicly until it has been addressed.

### How to Report

- Send an email to: security@asper-beauty-shop.com
- Include details about the vulnerability, including:
  - A description of the issue
  - Steps to reproduce the issue
  - Any relevant screenshots or logs

## Security Updates

We value security and will release updates regularly. We will also make
announcements for critical vulnerabilities through our GitHub repository and
other channels.

## Code Security Practices

- We adhere to best practices for secure coding and regularly update our
  dependencies to address potential vulnerabilities.
- Code reviews are conducted for all pull requests before merging.

## Acknowledgments

We thank all security researchers for their contributions and for responsibly
disclosing vulnerabilities to us.
